# Marcus voice rules

For the actual sound and rhythm of Marcus's voice, see `./voice-samples.md`. Read those samples first. This file is the operational rules that govern how voice gets applied to results-page copy specifically: required replacements, framings, and structural patterns the samples don't fully cover.

## Required replacements

If any of these phrases come up while writing copy, swap them for the version on the right. No exceptions.

| Don't say | Say instead |
|---|---|
| "There's a gap" | "You're not there yet" |
| "The math agrees" | "Your numbers point in that direction" |
| "You need" (for targets) | "Your current target is" |
| "That's a real distance" | "That's a meaningful shortfall" |
| "Most people end up with a gap" | "This is a common place to start" |

## Words to avoid

Score. Grade. Failing. Bad news. "You're behind" as a headline.

## Preferred framings (use these phrasings when they fit)

- "At your current pace, you're tracking toward..."
- "Based on what you entered..."
- "Your current target is..."
- "This is a common place to start."
- "Here's what would change that."

## Benchmark rule

Always translate benchmarks into plain English. Use "savings equal to about 100% of one year of income," not "1× income."

## Shortfall pattern (how to deliver hard news)

When the projected amount is below the target, follow this order:

1. State the projected amount.
2. State the current target.
3. Name it a meaningful shortfall.
4. Pivot to levers (save more, retire later, work part-time).

Example: *"At your current pace, you're tracking toward [projected]. Your current target is [target]. That's a meaningful shortfall. Here's what would move it."*

## Self-check before finalizing copy

- Read the line aloud. Does it sound like the voice samples in `./voice-samples.md`?
- Did I use any of the replacement phrases I shouldn't?
- Did I explain numbers in plain English?
- Would Marcus actually say this to a 42-year-old with two kids and a mortgage?
